# Relazione DineUp

File principali:

- `main.tex`: sorgente LaTeX;
- `main.pdf`: PDF compilato;
- `img/`: diagrammi UML e schema relazionale.

Compilazione consigliata su Overleaf: caricare l'intera cartella `relazione` e
impostare `main.tex` come documento principale.

In locale è possibile usare:

```text
tectonic main.tex
```

oppure una distribuzione LaTeX completa:

```text
latexmk -pdf main.tex
```
